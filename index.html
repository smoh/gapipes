Welcome to gapipes!¶

gapipes is a pandas-centric lightweight collection of routines to fetch and work with the Gaia data.

It provides

  • sensible and tested Gaia TAP+ client to fetch the data

  • custom accessors to pandas DataFrame and Series to do common operations on the Gaia data such as

    • making astropy coordinates objects

    • calculating renormalized unit weight error

    • making covariance matrix from errors and correlation coefficients.

A quick tour¶

Say you have some Gaia data. Let’s quickly fetch some from the Gaia archive.

import matplotlib.pyplot as plt
import gapipes as gp

df = gp.gaia.query("""
select top 50 *
from gaiadr2.gaia_source
where
1=contains(point('', ra, dec),
           circle('', 130.226, 19.665, 1))
and parallax between 4.613 and 7.312
""")

df is a pandas Dataframe containing gaia sources.

print(type(df), len(df), 'rows')

<class 'pandas.core.frame.DataFrame'> 50 rows

When you import gapipes, it registers a custom accessor g, under which there is a collection of common operations. Let’s just work with first 5 rows for display purpose.
