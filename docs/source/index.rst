Welcome to gapipes!
===================

**gapipes** is a collection of routines to fetch and work with `Gaia <#>`_ data.

It provides

- interface to `Gaia TAP+ <#>`_ and `DataLink Service <#>`_
- pandas `pipes <#>`_ to process Gaia DataFrames.
